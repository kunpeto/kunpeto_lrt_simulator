## 4. 模擬器實作要點

- 點擊駕駛室背景圖上的 DCI 區域 → 放大進入 DCI 全螢幕模式
- DCI 全螢幕模式中可操作通訊與廣播功能
- 點擊「返回」或螢幕外區域 → 縮小回駕駛室視圖
