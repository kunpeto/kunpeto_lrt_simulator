# CMS 螢幕規格

> 版本：v1.0
> 系統：CMS (Control and Monitor System) — Selectron 製造
> 參考照片：車號 111（2026-03-25 實拍）

---

## 1. 螢幕總體架構

CMS 螢幕分為 **6 個區域**，由上至下：

```
┌──────────────────────────────────────────────────┐
│  ① 訊息欄 Info Bar                                │
├──────────────────────────────────────────────────┤
│  ② 故障訊息欄 Error Bar                            │
├──────────────────────────────────────────────────┤
│  ③ 第三行資訊列                                    │
├────────────────────────┬─────────────────────────┤
│                        │                         │
│  ④ 主畫面左區           │  ⑤ 狀態顯示幕（右區）     │
│    （圖標＋功能按鈕）     │    （車門＋轉向架狀態）    │
│                        │                         │
├────────────────────────┴─────────────────────────┤
│  ⑥ 主選單欄 Menu Bar                               │
└──────────────────────────────────────────────────┘
```

## 2. 首頁（Home）畫面詳細規格

### 2.1 訊息欄 Info Bar（最上方）

左至右排列：

| 項目 | 格式 | 範例 | 說明 |
|------|------|------|------|
| 時間 | HH MM SS | 06 29 12 | 24 小時制，數字之間有空格 |
| 列車編號 | 列車編號：NNN | 列車編號：111 | 三位數車號 |
| 日期 | YYYY-MM-DD | 2026-03-25 | ISO 格式 |
| 車速 | N km/h | 0 km/h | 目前車速 |

右上角圖標列（由左至右）：
| 圖標 | 說明 | 亮燈條件 |
|------|------|---------|
| ⊕ 十字箭頭 | 方向/定位 | 常顯 |
| HB 圓圈 | 手煞車 | 手煞車作動中 |
| 照明圖標 | 頭燈狀態 | 頭燈開啟時 |

### 2.2 故障訊息欄 Error Bar

| 項目 | 格式 | 範例 |
|------|------|------|
| 資訊圖標 | ⓘ 圓圈 | 固定顯示於最左側 |
| 故障等級 | 三角警告圖標 | ▲ 紅色=嚴重 |
| 故障碼 | 數字 | 4610 |
| 時間戳記 | DD.MM.YYYY - HH:MM:SS | 25.03.2026 - 05:00:30 |
| 故障圖標 | 對應子系統 | ⚠ 牽引/煞車圖示 |
| 故障描述 | 中文文字 | 驅動/剎車控制C |

> 註：故障訊息欄僅顯示**最高優先權**的實際故障

### 2.3 第三行資訊列

| 項目 | 格式 | 範例 | 說明 |
|------|------|------|------|
| 里程數 | NNNNNN km | 310786 km | 列車總行駛里程 |
| 駕駛端 | 字母 | B | B=M5端 / A=M1端 / N=未啟動 |
| 運行模式 | 中文 | 主線模式 | 依行車模式旋鈕而定 |

運行模式對照表：
| 旋鈕位置 | CMS 顯示 |
|---------|---------|
| 關閉 | 關閉模式 |
| 主線 | 主線模式 |
| 機廠 | 機廠模式 |
| 洗車聯結 | 洗車聯結模式 |
| 倒車 | 倒車模式 |

### 2.4 主畫面左區 — 圖標區

圖標分為三排，背景為深色。圖標規則：**亮燈（淺色/白色背景）= 作動中**，**暗燈（灰黑）= 無作動**。

#### 第一排（系統狀態圖標）

| 位置 | 圖標 | 名稱 | 亮燈意義 | 暗燈意義 |
|------|------|------|---------|---------|
| 1 | 集電弓（弓形+線路） | 集電弓狀態 | 已升弓 | 已降弓 |
| 2 | 齒輪/煞車片 | 煞車系統 | 煞車作動中 | 正常待機 |
| 3 | (ABS) 圓圈 | 防鎖死煞車 | ABS 作動中 | 正常 |
| 4 | 空調風扇（前端） | 前端空調 | 運轉中 | 未運轉 |

#### 第二排（子系統狀態圖標）

| 位置 | 圖標 | 名稱 | 亮燈意義 | 暗燈意義 |
|------|------|------|---------|---------|
| 1 | 船型/轉向架 | 空轉/打滑 | 偵測到空轉或打滑 | 正常 |
| 2 | 火焰 | 火警偵測 | 偵測到煙霧/火警 | 正常 |
| 3 | 風扇（後端） | 後端空調 | 運轉中 | 未運轉 |
| 4 | ⚡ 110V | 110V 輔助供電 | 供電中 | 未供電 |
| 5 | ⚡ 380V | 380V 輔助供電 | 供電中 | 未供電 |

#### 第三排（功能按鈕，藍色底色可觸控）

| 位置 | 按鈕文字 | 功能 | 觸控效果 |
|------|---------|------|---------|
| 1 | 集電弓（三角/高壓）圖標 | 恢復集電弓自動升降 | 點擊觸發 |
| 2 | 輪緣潤滑啟動 | 手動啟動輪緣潤滑 | 藍色=啟用中 |
| 3 | 重置後視鏡 | 重置後視鏡位置 | 點擊觸發 |
| 4 | （空白區塊） | 預留 | — |
| 5 | 降弓 | 手動降弓 | 點擊觸發降弓指令 |

### 2.5 主畫面右區 — 車門＋轉向架狀態

此區域顯示列車 M1～M5 五個模組的車門狀態與轉向架資訊。

#### 頂部資訊
| 項目 | 範例 | 說明 |
|------|------|------|
| 電池電量 | -86 % | ESS 儲能電池電量百分比，負號表示放電中 |

#### 車門狀態圖示

```
         R     L
   M5   [R5]  [L5]    ← 每個方塊代表一扇車門
        T HB T         ← 轉向架標示（M5 轉向架）
   M4   [R4]  [L4]
                       ← M3 無車門（鉸接模組）
         HB            ← M3 區域標示（如 HB=手煞車）
   M2   [R2]  [L2]
        T HB T         ← 轉向架標示（M1 轉向架）
   M1   [R1]  [L1]
```

- 排列方向隨駕駛端翻轉：M1端駕駛→M1在下、M5端駕駛→M1在上
- 照片為 B 端（M5 端）駕駛，故 M5 在上方

#### 車門狀態顏色
| 顏色 | 狀態 |
|------|------|
| 亮色/藍色填滿 | 車門開啟 |
| 暗色/輪廓線 | 車門關閉 |

#### 轉向架標示符號
| 符號 | 意義 | 顯示位置 |
|------|------|---------|
| T | 牽引正常 (Traction) | 轉向架旁 |
| HB | 手煞車作動 (Hand Brake) | 轉向架旁 |
| PB | 駐車煞車作動 (Parking Brake) | 轉向架旁 |
| Se | 安全煞車作動 (Safety Brake) | 轉向架旁 + 右上角圓形圖示 |
| Err | 轉向架異常 | 紅色文字顯示於轉向架位置 |

## 3. CMS 子選單架構

### 3.1 設定（Settings）

| 子選單 | 功能 |
|--------|------|
| 語言＆亮度 | 中英雙語切換 + 螢幕亮度手動調整（+/−） |
| 狀況（或「網路」） | 指示器測試、墨砂測試、噴嘴加熱測試、手動輪緣潤滑 |
| 維護登錄 | 密碼輸入進入維護模式 |
| 軸徑設定 | — |
| 照明 | — |
| 其他 | — |

### 3.2 監視（Monitor）

| 子選單 | 功能 |
|--------|------|
| TCMS | CAN 匯流排網路拓撲架構，5 模組控制器在線狀態（藍色=正常，紅色=故障） |
| 電源 | ESS 電壓、750V 母線、110V/380V、24V 蓄電池、HSCB 狀態 |
| 容量 | 牽箱、輪緣潤滑、集電弓/牽引器圖示 |

### 3.3 控制（Control）

| 子選單 | 功能 |
|--------|------|
| 煙霧偵測器 | 11 組偵測器狀態，復置/後歸/隔離操作 |
| 客室空調 | 4 組空調（M4.2/M4.1/M2.2/M2.1）溫度與運作狀態 |
| 駕駛室空調 | 溫度設定（預設 24°C）、風速、新風/除霧/暖氣/低風組/關閉 |
| 車門 | 車門狀態與控制 |
| 推進 | 自動輪緣潤滑開關 + TCU 錯誤復位 |
| 煞車 | 轉向架選擇 → 煞車系統隔離（摩擦煞車/磁軌煞車） |

### 3.4 診斷（Diagnosis）

| 子選單 | 功能 |
|--------|------|
| DDS 故障紀錄 | 故障歷史：編號、時間、狀態、子系統、描述。等級篩選 0/A/B/C/D + 主動警報按鈕 |

## 5. 模擬器實作要點

### 5.1 CMS 觸控互動
- 點擊駕駛室背景圖上的 CMS 區域 → 放大進入 CMS 全螢幕模式
- CMS 全螢幕模式中可操作所有選單與按鈕
- 點擊「返回」或螢幕外區域 → 縮小回駕駛室視圖

### 5.2 首頁動態元素
| 元素 | 更新條件 |
|------|---------|
| 時間 | 即時更新（模擬時鐘） |
| 車速 | 隨主控制器檔位與物理模擬更新 |
| 故障訊息 | 觸發故障條件時顯示 |
| 車門狀態 | F1/F2/F3 操作後更新 |
| 轉向架標示 | 煞車/牽引狀態變化時更新 |
| 電池電量 | 模擬電量消耗 |
| 運行模式 | D3 行車模式旋鈕切換時更新 |
| 駕駛端 | 固定為 A（M1 端模擬） |
| 圖標亮暗 | 對應子系統狀態變化 |

### 5.3 開機動畫
KEY ON 後 CMS 開機順序：
1. 螢幕全黑 → 亮起
2. Selectron 品牌標誌（暫定）
3. 系統自檢畫面
4. 進入首頁
