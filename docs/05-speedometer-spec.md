# 車速表規格

> 版本：v0.1（骨架，待補）
> 設備：雙針時數表（類比式）
> 設備編號：A5
> 位置：上層弧形面板中央，圓形綠框

---

## 1. 外觀規格

| 項目 | 說明 |
|------|------|
| 形狀 | 圓形錶面，綠色邊框 |
| 尺寸 | （待量測實際直徑） |
| 刻度範圍 | 0 ~ 90 km/h |
| 指針 | 白針（目前車速）、紅針（速限） |

---

## 2. 指針行為

### 2.1 白針（目前車速）

| 狀態 | 行為 |
|------|------|
| 停車時 | 指向 0 |
| 加速時 | 隨車速順時針偏轉 |
| 減速時 | 隨車速逆時針回轉 |

### 2.2 紅針（速限）

| 狀態 | 行為 |
|------|------|
| 正常行駛 | 指向目前區段速限值 |
| 速限變更 | 依 ATP 訊號調整 |

---

## 3. 自檢動畫

KEY ON 後車速表進行自檢：

```
白針動畫序列：0 → 50 → 90 → 0 km/h
```

- 自檢時間：約 3~5 秒
- 自檢完成後指針回到 0，表示正常
- 自檢未通過（指針卡住）需重新啟動

---

## 4. 模擬器實作要點

- 使用 SVG 或 Canvas 繪製圓形錶面與指針
- 指針旋轉角度 = 車速值映射到角度範圍
- 自檢動畫使用 CSS animation 或 requestAnimationFrame
- 車速由主控制器檔位 + 物理模擬引擎計算
