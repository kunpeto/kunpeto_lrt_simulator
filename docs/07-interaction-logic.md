## 1. 啟動程序連鎖邏輯

### 1.1 發車程序

```
KEY ON (D1)
  │
  ├─→ 車速表自檢：白針 0→50→90→0
  ├─→ CMS 開機動畫：黑屏→品牌標誌→自檢→首頁
  ├─→ DCI 開機
  │
  ▼
行車模式旋鈕 (D3) → 切離「關閉模式」
  │
  ▼
按住停機連鎖 (D2) + 主控制器 (D4) 推離 STOP
  │
  ├─→ CMS 四個 T 亮起（牽引就緒）
  │
  ▼
安全煞車測試
  │
  ├─→ 按下 B1 安全煞車
  ├─→ CMS 顯示 Se
  ├─→ 旋轉 B1 復歸
  ├─→ Se 消失 → 測試通過
  │
  ▼
系統就緒，可出發
```

### 1.2 收車程序

```
車輛停妥，車速歸零
  │
  ▼
按住停機連鎖 (D2) + 主控制器 (D4) 拉回 STOP
  │
  ▼
行車模式旋鈕 (D3) → 關閉模式
  │
  ▼
KEY OFF (D1)
```

## 2. 主控制器狀態機

### 2.1 檔位定義

```
STOP ←─(需 D2)─→ EB ←─(過卡檔)─→ SB(0~100%) ←→ N ←→ D(0~100%)
```

### 2.2 狀態轉移規則

| 從 | 到 | 條件 |
|----|-----|------|
| STOP | EB | 按住 D2 停機連鎖 |
| EB | STOP | 按住 D2 停機連鎖 |
| EB | SB | 推過卡檔 |
| SB | N | 推至頂部 |
| N | D | 繼續往上推 |
| D | N | 放手自動回 N |
| N | SB | 往下拉 |

## 4. 故障模擬情境

### 4.1 計劃支援的故障情境

| 情境 | 觸發方式 | 預期表現 |
|------|---------|---------|
| 牽引故障 | （待定） | 總故障燈亮紅，CMS 顯示故障碼 |
| ATP 介入煞車 | 超速時自動觸發 | 車輛自動煞停，需按 Ruck 復歸 |
| 車門故障 | （待定） | CMS 車門圖示紅叉 |
| 集電弓異常 | （待定） | CMS 集電弓圖標閃爍 |
| 客室 PEB 觸發 | 模擬旅客按壓 | B2 亮燈，CCTV 自動鎖定 |

### 4.2 重開機程序

| 層級 | 操作 | 適用情境 |
|------|------|---------|
| WD0 | CMS 觸控畫面重開機 | CMS 觸控異常 |
| WD1 | KEY OFF → 30秒 → KEY ON | 一般故障復歸 |
| WD2 | 24V 斷電 → 30秒 → 24V 復電 | 較嚴重故障 |
| WD3 | 降弓 → 全車斷電 → 復電 | 最嚴重故障 |
