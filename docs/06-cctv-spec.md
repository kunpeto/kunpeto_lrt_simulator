# CCTV 螢幕規格

> 版本：v0.1（骨架，待補）
> 設備編號：A1（主 CCTV 螢幕）、C1（後視鏡 CCTV）

---

## 1. 系統架構

### 1.1 主 CCTV 螢幕（A1）

| 項目 | 說明 |
|------|------|
| 位置 | 上層弧形面板最左側 |
| 類型 | 灰色小螢幕 |
| 攝影機數量 | 16 台 |
| 顯示模式 | 2×2 循環模式 / 目錄模式 |
| 特殊功能 | 緊急事件（PEB）自動鎖定對應攝影機 |

### 1.2 後視鏡 CCTV（C1）

| 項目 | 說明 |
|------|------|
| 位置 | 上層最右側小窗 |
| 攝影機數量 | 左右各 2 台（共 4 台） |
| 獨立性 | 獨立系統（不經網路） |
| 觸發條件 | 按下車門授權按鈕時顯示 |
| 顯示內容 | 第一道車門下方旅客進出狀況 |

## 3. 操作互動

| 操作 | 效果 |
|------|------|
| 循環模式 | 自動輪播 4 組 2×2 畫面 |
| 目錄模式 | 手動選擇攝影機 |
| PEB 觸發 | 自動鎖定對應車廂攝影機 |

---

## 4. 模擬器實作要點

- 點擊 A1 區域 → 放大進入 CCTV 全螢幕
- 模擬畫面可用靜態截圖或色塊替代
- C1 後視鏡在車門授權時自動顯示
