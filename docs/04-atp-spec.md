# ATP 速度監控 HMI 規格

> 版本：v0.1（骨架，待補）
> 系統：ATP (Automatic Train Protection) 速度監控人機介面
> 設備編號：A6
> 位置：車速表上方小螢幕

---

## 1. 螢幕總體架構

ATP HMI 顯示列車速限資訊與 ATP 系統狀態，是司機員判斷速限的主要依據。

```
┌──────────────────────┐
│  （畫面佈局待補）      │
│                      │
└──────────────────────┘
```

---

## 2. 主要顯示項目

| 項目 | 說明 |
|------|------|
| 目前速限 | 數字顯示目前區段允許最高速度 |
| 目標速限 | 前方減速目標（如有） |
| ATP 狀態 | 正常監控 / 介入煞車 / 釋放 |

---

## 3. 操作按鈕

| 按鈕 | 英文 | 功能 |
|------|------|------|

## 5. 模擬器實作要點

- 點擊駕駛室背景圖上的 ATP 區域 → 放大進入 ATP 面板
- 速限數字依模擬場景動態更新
- Ruck 按鈕在 ATP 介入煞車後可操作
